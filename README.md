## 환경변수 설정

아래 값들은 각자 발급받은 API Key로 설정해야 합니다.

```powershell
setx OPENAI_API_KEY "your_openai_api_key"
setx NAVER_CLIENT_ID "your_naver_client_id"
setx NAVER_CLIENT_SECRET "your_naver_client_secret"
setx TELEGRAM_BOT_TOKEN "your_telegram_bot_token"
setx TELEGRAM_CHAT_ID "your_telegram_chat_id"
